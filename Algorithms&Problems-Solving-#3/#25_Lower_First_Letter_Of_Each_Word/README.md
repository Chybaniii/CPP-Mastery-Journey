# Problem 25 - Lower First Letter of Each Word

## Description:
This program reads a full string from the user and converts the first letter of each word to lowercase.

## Concepts Used:
- Strings in C++
- For loop
- Boolean flag (IsFirstLetter)
- Conditions (if statements)
- Character manipulation
- toupper()

## Notes:
- I learned how to detect the beginning of each word
- The program checks every character in the string
- When a space is found, the next character becomes the first letter of a new word
