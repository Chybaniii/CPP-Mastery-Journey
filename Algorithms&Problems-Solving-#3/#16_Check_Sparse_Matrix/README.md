# Problem 16 - Check Sparse Matrix

## Description:
Fill a 3x3 matrix with values hard-coded and print it.
Then Check if the matrix is sparse or not, And print the result..

## Concepts Used:
- 2D Arrays
- Nested for loops
- Boolean function
- Conditional statements

## Notes:
- In this problem, I learned how to check if a matrix is sparse or not.

## Difference Between Solutions:
- In the first approach, I used only one function to check the sparse matrix
Without extra functions and, I used counters to check whether the number of zeros 
Are greater than the Number of non-zeros elements, and print the result.

- The second approach is similar to the first one the only difference is
'CountNumberInMatrix' function is a function which is responsible for returning the Count 
I used it to return the count of zero to compare it with Number of elements in matrix .
I also used an 'IsSparseMatrix' bool function to retrun whether the result is true or false.
