# Problem 06 - 3x3 Ordered Matrix

## Description:
Fill a 3x3 matrix with order numbers and print it.

## Concepts Used:
- 2D Array.
- Nested for loops
- Static Variables

## Notes:
- I used a static variable    inside the ' OrderNumbers() ' function to generate orederd Numbers in the  matrix.


- Another approach uses a simple counter variable and increments it 
In each iteration.

- Using a static variable here may cause the counting to continue
Instead of starting from 1 if the function is called again.
