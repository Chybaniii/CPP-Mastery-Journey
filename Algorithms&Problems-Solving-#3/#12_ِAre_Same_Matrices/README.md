# Problem 12 - Are Typical/Same Matrices

## Description:
- Fill a two 3x3 matrices and print it.
- Compare two matrices and check if they are identical.
- Then print whether the matrices are identical or not.

## Concepts Used:
- 2D arrays
- Random numbers generation
- Function
- Boolean function
- Conditional statements

## Notes:
- In this problem, I learned How to compare matrices element by element.

- The Solution use direct comparison.

## Difference Between Solutions:
- Both solutions use the same logic to compare
 matrices.

- The difference is in the structure of the function:
  the comparison function checks each element one by one and returns false immediately if a mismatch is found, otherwise returns true.
