# Problem 20 - Palindrome Matrix

## Description
Fill a 3x3 matrix with hard-coded values and print it.

Then check whether the matrix is palindrome or not.

## Concepts Used
- 2D Arrays
- Nested loops
- Functions
- Comparing elements

## Notes
- In this problem, I learned how to compare elements from both sides of each row to check whether the matrix is palindrome or not.
