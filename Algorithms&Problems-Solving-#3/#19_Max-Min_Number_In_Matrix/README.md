# Problem 19 - Maximum and Minimum Number in matrix

## Description:
Fill a 3x3 matrix with hard-coded 
Values and print it.

## Concepts Used:
- 2D arrays
- Nested loops
- Functions
- conditional statements

## Notes:
- In this problem, I learned how to find the minimum and maximum
values in a matrix using nested for loop.

- This problem is similar to the Min\Max problem 
In 1D arrays, but applied to matrice.
