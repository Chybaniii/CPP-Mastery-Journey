# Problem 03 - Sum Each Row in Matrix in Array

## Description:
Fill a 3x3 matrix with random numbers, then calculate
and store the sum of each row in a separate array.

## Concepts Used:
- 2D Arrays
- 1D Arrays
- Nested for loops
- Random number generation
- Function Decomposition

## Notes:
- I used a separate 1D array to store the sum of each row.
- Each function has one job only (Single Responsibility).
- I used a function to calculate the sum and another
  to store it, and another to print it.
