
# ✊✋✌️ Rock Paper Scissors

A console-based Rock Paper Scissors game built 
with C++ where you play against the computer.

## How it works

You choose the number of rounds (1 to 10),
then each round you pick Stone, Paper, or Scissors.
The computer picks randomly, and the winner 
of each round is shown immediately with colors:

- 🟢 Green = You won
- 🔴 Red   = Computer won  
- 🟡 Yellow = Draw

The game tracks all rounds and shows final results.

## What I built

- Round-by-round winner detection
- Score tracking across all rounds
- Screen color changes based on winner
- Play again option at the end

## Built with

- Language: C++
- Concepts: Enums, Structs, Functions, Switch

## How to run

Compile and run in any C++ environment.
