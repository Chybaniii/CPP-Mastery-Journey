# 🧮 Math Game

A console-based math quiz game built with C++.
Test your math skills!

## How it works

At the start you choose:
- Number of questions (1 to 10)
- Difficulty level: Easy / Mid / Hard / Mix
- Operation type: Add / Sub / Mul / Div / Mix

Each question shows two numbers and an operation.
Type your answer and see if you got it right.

- 🟢 Green screen = Correct answer
- 🔴 Red screen + beep = Wrong answer

At the end you get your final score and Pass or Fail.

## How to exit

After each game you will be asked:
"Do you want to play again? Y/N"
- Type Y or y → play again
- Type any other key → game exits

## What I built

- Random question generator by difficulty
- Questions stored in an Array of 100 slots
- Auto score tracking
- Screen color changes per answer
- Pass/Fail result at the end

## Built with

- Language: C++
- Concepts: Enums, Structs, Arrays, Functions, Switch
